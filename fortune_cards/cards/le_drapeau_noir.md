# Le Drapeau Noir

## Règle
Si vous marquez **800 points ou plus** ce tour, vous recevez **500 points bonus** supplémentaires. En dessous de 800 points (ou si vous êtes éliminé), votre score est normal (ou 0).

## Stratégie induite
Viser agressivement les grosses combinaisons plutôt que de s'arrêter prudemment sur un score modeste ; 800 pts est un seuil atteignable mais qui demande de prendre des risques.
